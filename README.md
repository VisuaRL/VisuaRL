# VisuaRL 
A tool to teach reinforcement learning.

[Link to try](http://vrl-test.azurewebsites.net/)

Note that it might take awhile for the training to load - don't panic!

Frameworks used:
- react
- redux
- redux toolkit
- joyride
- react-hook-form

`npm install`
To install dependencies.

`npm start`
To start development server.
